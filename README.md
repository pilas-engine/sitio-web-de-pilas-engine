## ¿Cómo iniciar el servidor?

Primero tenes que asegurarte de tener instaladas estas dependencias:

```
sudo gem install bundler jekyll
```

Luego, para iniciar el proyecto y el servidor:

```
make iniciar
make ejecutar
```

o bien:

```
jekyll s -l
```
